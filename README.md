# Basic Node.js HTTP Servers

This project contains two simple examples of how to create a basic web server using Node.js's built-in `http` module.

---

## Files Included

###  `basic-server.js`

- Responds with a plain text message: `Hello from Node.js Server`
- Always returns 200 OK
- Runs on port `3000`

```bash
node basic-server.js
